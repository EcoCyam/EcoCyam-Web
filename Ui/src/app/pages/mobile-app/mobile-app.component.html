Fonctionnalités

Analyse électroménager

Ecocyam analyse les produits électroménager. Vous accédez aux détails de la composition du produit pour comprendre son évaluation.

Analyse smartphone

Ecocyam analyse vos smartphones et vous explique l'évaluation de chaque produit dans une fiche produit détaillée.

Recommandations

Pour les produits médiocres ou mauvais que vous avez scannés, Ecocyam recommande en toute indépendance des produits similaires meilleurs pour l'environnement.

Historique

Ecocyam affiche l'historique des produits scannés. Vous identifiez l'impact de chaque produit sur l'environnement via un code couleur simple et des diagrammes
